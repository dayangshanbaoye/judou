Design Principles
本地优先
所有数据（EPUB、断句、卡片、音频）存本地 SQLite。仅「主动送去讲解/合成」的文本片段出网。
可追溯
任意句子都能回答「属于哪章哪节哪段」。面包屑 + 硬外键约束保证追溯链不断裂。
结构化输出
LLM 讲解固定四段 JSON：语法 / 难点词 / 翻译 / 通俗表达。可存储、可搜索，不做自由文本。
ICAP 闭环
读（P）→ 标记/讲解（A）→ 造句（C）→ 复习（C）。每功能对应 ICAP 层级，不做「只读不练」。
